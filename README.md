# AUTH Fullstack app

Only autorization with pair of tokens (access and refresh).

## Stack

### Server side

- Node.JS (Express), TypeScript
- **DB:** PostgresQl
- Express Validator

### Client side

- React.JS, TypeScript
- **Tanstack Query** - server requests
- **React Hook Form** - form validation
- **TailWind** - styling
- **Zustand** - for state managment
- **Vitest / React Testing Library** - for testing

## How to run

### 1. To start the server

`npm run server`

### 2. To start the client

`npm run client`

### 3. To start the server with client

`npm run dev`
